# Shooter Web

Przegladarkowy prototyp dynamicznej strzelanki 2D z deformowalnym terenem.

## Lokalnie

```bash
node server.js
```

Potem otworz `http://localhost:3000`.

## GitHub Pages

Repo jest przygotowane do publikacji jako GitHub Pages.

Opcja 1:
- wrzuc `shooter-web` jako osobne repo
- push na branch `main`
- w ustawieniach GitHub włącz `Pages` z `GitHub Actions`

Opcja 2:
- jeśli trzymasz ten katalog wewnatrz wiekszego repo, przenies:
  - zawartosc `docs/`
  - katalog `.github/workflows/`
  do roota repo, z ktorego chcesz publikowac stronę

Publikowana wersja strony siedzi w `docs/`.
